# SMTH Admin Front
Клиентская часть админ-панели **SMTH**

## Установка
0. **Зависимость!** Установить https://github.com/4ashk1n/smth-shared в ту же папку, в которой будет папка с smth-front-admin
1. `git clone https://github.com/4ashk1n/smth-front-admin.git`
2. `cd smth-front-admin`
3. `cp .env.example .env`
4. Настроить переменные окружения в `.env`
5. `npm install`

## Запуск
1. Dev-mode: `npm run dev`
2. Сборка: `npm run build`
3. Превью билда: `npm run preview`
